# Big Game Census Analytics

## Overview
Big Game Census Analytics is a data-driven project designed to analyze and visualize census data related to Super Bowl 52 players. This project leverages data visualization techniques to gain insights into player distribution, demographics, and key performance indicators.

## Features
- **Data Processing & Cleaning**: Prepares raw census data for analysis.
- **Interactive Dashboards**: Tableau-based visualizations for intuitive data exploration.
- **Insights & Reports**: Provides key findings on player demographics and distribution.

## Files in Repository
- **BigGameCensusAnalytics.ipynb**: Jupyter Notebook containing data analysis and visualization.
- **Big_Game_Census_Dashboard.twb**: Tableau workbook for interactive data visualization.

## Installation & Usage
### Prerequisites
Ensure you have the following installed:
- Python (>=3.7)
- Jupyter Notebook
- Pandas, NumPy, Matplotlib, Seaborn
- Tableau (for dashboard visualization)

### Running the Project
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/BigGameCensusAnalytics.git
   cd BigGameCensusAnalytics
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the Jupyter Notebook:
   ```bash
   jupyter notebook BigGameCensusAnalytics.ipynb
   ```
4. Open Tableau and load `Big_Game_Census_Dashboard.twb` to explore the interactive dashboard.

## Results & Insights
- There are **58 players** in Philadelphia and **60 players** in New England.

- The highest number of players come from **California**, followed by **Texas**.
- **New England team** has more players from **Texas**, while **Philadelphia team** has more players from **California**.
- The **average age** of New England players is **26.967 years**, while for Philadelphia, it is **26.465 years**.
- The **average weight** of New England players is **244.983 lbs**, while for Philadelphia, it is **242.741 lbs**.
